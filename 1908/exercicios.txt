EX1: Construa um algortimo que inverta um vetor de N posições, sem usar vetor auxiliar. (N)

inverte_vetor(*vetor, n): vetor {

	for (i=0; i < n/2;i++) {
		aux = vetor[i]
		vetor[i] = vetor[n-i]
		vetor[n-i] = aux
	}
}
